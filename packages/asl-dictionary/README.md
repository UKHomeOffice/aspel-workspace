# asl-dictionary

Expansions of acronyms commonly used in ASL projects


## Usage

Install the package:

```bash
npm install @asl/dictionary --save-prod
```

Import as necessary:

```js
const { NACWO, NTCO } = require('@asl/dictionary');
console.log(NACWO);
```

result:
```bash
Named Animal Care and Welfare Officer
```
